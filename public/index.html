Welcome to the TikTok Data Retrieval API

API Endpoint

GET /api/tiktok

Parameters

  • username (required) - TikTok username (without @)
  • page (optional) - Page number (default: 1)
  • per-page (optional) - Posts per page (default: 10, max: 100)
  • start_epoch (optional) - Filter videos after this timestamp
  • end_epoch (optional) - Filter videos before this timestamp

Example Request

GET /api/tiktok?username=example&page=1&per-page=10
